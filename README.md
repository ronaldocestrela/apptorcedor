# FFC App Torcedor - Versão Web

Versão web de apresentação do aplicativo do Feira Futebol Clube, baseada nos designs mobile.

## Telas

- **Notícias** – Lista de artigos com thumbnail, título e resumo; clique para ver detalhe
- **Detalhe da Notícia** – Imagem hero, título e texto completo com botão voltar
- **Loja** – Grid de produtos com preço e botão "Adicionar ao Carrinho"
- **Perfil** – Avatar, dados do sócio e seção de prêmios/badges

## Stack

- React 18 + TypeScript
- Vite
- Tailwind CSS
- React Router v6
- Lucide React (ícones)

## Desenvolvimento

```bash
npm install
npm run dev
```

Acesse http://localhost:5173

## Build

```bash
npm run build
```

Arquivos gerados em `dist/`.
